# OceanBase SIG
OceanBase Database 是一个分布式关系型数据库。完全由蚂蚁集团自主研发。 OceanBase 基于
 Paxos 协议以及分布式架构，实现了高可用和线性扩展。OceanBase 数据库运行在常见的服务>器集群上，不依赖特殊的硬件架构。

2021年6月1日，OceanBase正式开源，包括服务端，客户端，工具链，依赖库等，并为开源爱好者参与OceanBase提供支持。
