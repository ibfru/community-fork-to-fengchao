
# sig-compat-winapp

compat-winapp SIG组致力于将X86平台下海量的 Windows 应用引入X86/ARM等架构的 openEuler 系统中，通过提供完善的构建解决方案和指引文档，让更多对此感兴趣的人加入其中。

# 组织会议

- 公开的会议时间：每两周一次例会，例会时间每周三下午 15:00 - 15:30 +0800 北京时间
