# SIG名称

Haskell是一种函数式、静态、隐式类型、惰性的编程语言。Haskell 已经是一种广泛使用的语言，基于 Haskell 开发的不少组件也已经得到广泛应用。本 SIG 关注将 Haskell 语言的工具及相关软件组件生态引入到 openEuler 之上。
